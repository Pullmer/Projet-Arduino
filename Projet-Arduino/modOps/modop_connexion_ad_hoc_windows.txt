Ce modop explique comment se connecter à un réseau ad-hoc depuis windows.

1) Créer une nouvelle connexion réseau dans Panneau de configuration\Réseau et Internet\Centre Réseau et partage

2) Choisir "Se connecter manuellement à un réseau sans fil"

3) 	"RPi" pour le nom (SSID du réseau ad-hoc)
	Type de sécurité : WEP
	Ne pas oublier le mot de passe
	Décocher "Lancer automatiquement cette connexion"

4) Il faut maintenant modifier la configuration pour que Windows le reconnaisse comme du ad-hoc

Lancer un invité de commande administrateur (clic droit sur le logo Windows)

Lancer les commandes suivantes :

DOS:netsh wlan show profiles

Vérifier que le profile "RPi" apparait

DOS:netsh wlan set profileparameter name=RPi ConnectionType=IBSS

Vérifier les réseaux accessibles :

DOS:netsh wlan show networks

Connectez vous avec :

DOS:netsh wlan connect name=RPi

Vérifiez que vous êtes bien connectés dans le Centre de Réseau et Partage.
